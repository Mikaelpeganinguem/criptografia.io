Cifra de Vigenére

COMO USAR? Você pode criptografar, ou decriptografar as suas mensagens. Nessa cifra você preenche assim:
Mensagem: Coloque a palavra que você quer criptografar ou a palavra criptografada.
Senha: Digite uma palavra que será a sua senha. Nesse método NÃO PODE USAR NÚMEROS COMO SENHA. Em vez de números, use qualquer palavra/letra como acesso para criptografar/decriptografar.
Logo após, use o método que preferir e terá o resultado abaixo tanto da palavra criptografada, quanto decriptografada
OBS: Para cada criptografia/decriptogtafia, clique em: 'Nova criptografia'
